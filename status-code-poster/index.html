HTTP Status Code Poster

This nifty print-it-yourself poster is a great reference for developers to hang near their workspace. For best results, follow the directions provided below.

Size 19.75x22 inches

Directions

  1. Click the download button above.
  2. Send the file to your local print shop and inform them that there is a 0.5” bleed (they’ll know what you mean).
    Optional: ask for it in matte or gloss.
  3. Pick up your new poster.
  4. Hang your poster on your wall above your computer workspace.
